# pdx taxes

A free, frictionless guide to local taxes in Portland, Multnomah County, and Metro.

This site exists because Portland-area local taxes are scattered across multiple websites with different deadlines, account systems, and filing flows. We collect everything in one place: what you owe, when it's due, and where to file.

**This is an unofficial community resource. It is not legal, tax, or financial advice.**

## Develop

```bash
npm install
npm run dev
```

Open <http://localhost:4321> for the English site and <http://localhost:4321/es> for Spanish.
